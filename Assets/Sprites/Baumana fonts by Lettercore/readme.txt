-------

Baumana is a set of 3 pixel fonts that was very fun to design and very hard to explain.

Baumana Numera is a simple font based on retro style display. It contains only numbers and some math. It's very small and can be used in UI design. Baumana Numera is monospaced.

Baumana Code is a cryptic font-like entity born out of Baumana Numera. It's a set of all 128 variations of an 8-shaped segmented base that resembles a number on retro style display. Every glyph is unique and is placed somewhere in the vicinity of Basic Latin area of Unicode with some exceptions that are just magically elsewhere. There's a table with Unicode codes included to make sense of where everything is. Baumana Code is monospaced.

... and Baumana Litera is an experiment. It's an attempt to make a full font out of those 128 variants. It contains some Basic Latin, some Basic Cyrillic, and some numbers with math. A decent part of it is legible, lots of glyphs are repeated. Baumana Litera was supposed to be monospaced but it's not because some glyphs are twice the normal width.

The set also includes a bitmap version of Baumana Code that naturally contains two other fonts as well.

These fonts are free for personal use. Let me know if you like them or if you use them somewhere. If you like them or use them please consider paying for them.

Baumana – это набор из 3 шрифтов, которые было интересно рисовать, но сложно объяснить.

Baumana Numera – это простой шрифт примерно как на старом дисплее. В нём только цифры и немного математики. Он маленький и подойдёт для дизайна интерфейсов. Моноширный.

Baumana Code – это нечто шифроподобное в виде шрифта, рождённое из Baumana Numera. Это набор из всех 128 вариантов отображения сегментов базовой восьмёрки на старом дисплее. Каждый символ уникален и встречается в шрифте только один раз. Всё это расположено в секторе базовой латиницы юникода, но некоторые символы находятся в других места, потому что не поместились в базовую латиницу. Чтобы понять что где, прилагается таблица с кодами юникода. Шрифт моноширный.

... и Baumana Litera – это эксперимент. Это попытка сделать полноценный шрифт на базе предыдущих 128 символов. В нём есть русский и английский алфавиты плюс цифры, знаки препинания и математические операторы. Внушительная часть шрифта читаема, многие символы повторяются. Шрифт должен был быть моноширным, но не вышло, так что некоторые символы имеют двойную ширину, а один даже тройную.

Ещё в наборе лежат битмапы для Baumana Code, которые волшебным образом включают в себя и два других шрифта.

Эти шрифты бесплатны для личного использования. Если используете где-то, скажите мне, я хочу посмотреть. Если вы используете эти шрифты и они вам нравятся, то подумайте над оплатой.
